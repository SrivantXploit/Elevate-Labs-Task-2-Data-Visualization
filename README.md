# Elevate Labs - Task 2: Data Visualization and Storytelling

## Objective
Create an interactive sales dashboard using Tableau to visualize sales data and derive business insights.

## Dashboard Visualizations
- Sales by Product Line
- Sales by Country
- Sales by Deal Size
- Monthly Sales Trend
- Top 10 Customers

## Business Insights
- Classic Cars generated the highest sales.
- USA recorded the highest overall sales.
- Medium-sized deals contributed the highest revenue.
- Sales peaked during November.
- Euro Shopping Channel was the top customer by sales.

## Tools Used
- Tableau Desktop
- CSV Dataset
